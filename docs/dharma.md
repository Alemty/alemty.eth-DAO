
# Dharma — Sistema de Experiencia y Progreso  
alemty.eth · Documento Canónico v2.0

---

## 1. Definición

**Dharma** representa la **experiencia acumulada** de un usuario dentro del ecosistema alemty.eth.

El Dharma:
- Es equivalente a **XP**
- Representa aprendizaje, constancia y aportación real
- Se acumula de forma permanente
- Determina la progresión de rangos

> El Dharma sigue el principio budista:  
> se construye con acciones, no se compra.

---

## 2. Propiedades del Dharma

- 1 Dharma = 1 punto de experiencia (XP)
- El Dharma:
  - ✅ Se gana participando
  - ✅ Se acumula
  - ❌ No se compra
  - ❌ No se intercambia
  - ❌ No se dona
- El Dharma:
  - Llena una **barra de progreso**
  - Al completarse → **sube el rango**

---

## 3. Cómo se obtiene Dharma

Un usuario obtiene Dharma cuando otros usuarios interactúan positivamente con su contenido:

- Likes recibidos
- Comentarios valorados
- Puntajes otorgados (+1 a +10)
- Interacciones significativas

Ejemplo:
- Recibir 1 like → +1 Dharma
- Recibir +5 puntos en un post → +5 Dharma

---

## 4. Dharma y Rangos

El Dharma es el **único factor** que determina el rango de un usuario.

- No existen atajos
- No existe compra de experiencia
- No existe conversión de tokens a Dharma

> **No hay Pay‑to‑Level. Nunca.**

---

## 5. Relación Dharma ↔ Aura

Por cada punto de Dharma generado:
- Se genera **1 punto de Aura**

Esta generación es automática y unidireccional.

- Dharma = mérito
- Aura = valor

---

## 6. Principio Final

> El Dharma se cultiva.  
> El rango se alcanza.  
> La experiencia no se negocia.

alemty.eth protege la meritocracia como pilar del sistema.
