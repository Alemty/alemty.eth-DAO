
# Token Utility (alemty.eth)

> No es inversión. No representa equity. No promete retorno.

## Propósito
El token existe para coordinar comportamiento, acceso, reputación y gobernanza dentro del ecosistema.

## Reglas duras
- No ICO
- No preventa
- No emisión masiva inicial
- Emisión gradual basada en actividad

## Distribución (límite contable)
- Comunidad: 50%
- Reserva ecosistema: 20%
- Equipo/fundador: 15%
- Pool DAO/gobernanza: 10%
- Eventos/activaciones: 5%

## Uso
- Desbloqueos de funciones
- Accesos
- Incentivos no especulativos
- Consumo/quema parcial según acciones
